Nuestra Oferta Educativa

Paquete Completo de Office

Domina las herramientas esenciales de Microsoft Office (Word, Excel, PowerPoint) para optimizar tu productividad en cualquier entorno académico o profesional, creando documentos, hojas de cálculo y presentaciones impactantes.

  • Word: Creación y formato avanzado de documentos, uso de estilos, plantillas, combinación de correspondencia, tablas e ilustraciones.
  • Excel: Introducción a fórmulas y funciones, formato de celdas, creación de gráficos, tablas dinámicas básicas, ordenación y filtrado de datos.
  • PowerPoint: Diseño efectivo de diapositivas, uso de temas y patrones, inserción de multimedia, animaciones, transiciones y consejos para presentaciones.

Desarrollo Web Full Stack

Construye aplicaciones web interactivas y funcionales desde cero. Aprende las bases del frontend con HTML5, CSS3 (incluyendo Flexbox y Grid para diseño responsive) y JavaScript moderno (ES6+), y maneja datos con bases de datos relacionales usando SQL.

  • HTML5: Estructura semántica del documento, etiquetas esenciales, formularios y elementos multimedia.
  • CSS3: Selectores, modelo de caja, Flexbox para layouts flexibles, Grid para maquetación avanzada, diseño responsive y transiciones básicas.
  • JavaScript (ES6+): Variables, tipos de datos, operadores, estructuras de control, funciones, manipulación del DOM y eventos.
  • SQL: Conceptos de bases de datos relacionales, consultas básicas (SELECT, INSERT, UPDATE, DELETE), cláusulas WHERE, JOINs simples.

Fundamentos de Programación

Iníciate en el apasionante mundo del código desarrollando una sólida base lógica. Aprende a resolver problemas usando pseudocódigo (Pseint) y da tus primeros pasos en lenguajes demandados como Python y C++, además de entender cómo interactuar con bases de datos SQL.

  • Lógica y Algoritmos: Introducción al pensamiento computacional, diagramas de flujo y pseudocódigo utilizando Pseint para practicar.
  • Python Básico: Sintaxis fundamental, variables, tipos de datos, estructuras de control (if, for, while), funciones básicas.
  • C++ Básico: Introducción a la sintaxis, tipos de datos, entrada/salida estándar, estructuras de control, compilación básica.
  • SQL Básico: Comprensión de tablas y relaciones, ejecución de consultas SELECT sencillas para recuperar información.

Manejo Experto de Windows

Saca el máximo provecho al sistema operativo más utilizado. Aprende a navegar eficientemente, organizar tus archivos y carpetas, personalizar tu entorno, utilizar herramientas del sistema y aplicar buenas prácticas de seguridad.

  • Interfaz y Navegación: Escritorio, barra de tareas, menú Inicio, explorador de archivos (crear, mover, copiar, eliminar archivos/carpetas), vistas y búsqueda.
  • Organización y Personalización: Creación de bibliotecas, configuración del escritorio, opciones de pantalla, cuentas de usuario básicas.
  • Herramientas del Sistema: Panel de Control vs. Configuración, instalación/desinstalación de programas, herramientas básicas de mantenimiento (liberador de espacio).
  • Conceptos Básicos: Uso del portapapeles, atajos de teclado comunes, conexión a redes Wi-Fi, nociones de seguridad (antivirus, actualizaciones).

Herramientas de Inteligencia Artificial

Descubre cómo la Inteligencia Artificial puede potenciar tu creatividad y eficiencia. Aprende a utilizar herramientas líderes como ChatGPT, Gemini y Copilot para generar texto, obtener ideas, programar, editar imágenes y optimizar tus tareas diarias.

  • Generación de Contenido: Uso efectivo de ChatGPT y Google Gemini para crear textos, resumir información, brainstorming y responder preguntas.
  • Asistencia en Código y Tareas: Cómo usar GitHub Copilot (o similares) para autocompletar código, sugerir soluciones y depurar errores básicos.
  • Edición de Imágenes con IA: Introducción a herramientas online (como PIXLR AI, Canva Magic Studio) para mejorar fotos, eliminar fondos o generar elementos gráficos.
  • Prompts Efectivos: Aprende a formular preguntas e instrucciones claras para obtener los mejores resultados de las IAs generativas. Casos de uso prácticos.

Edición de Video Profesional

Aprende a crear videos atractivos y de alta calidad utilizando los programas líderes del mercado: Camtasia, Filmora y Sony Vegas. Ideal para creadores de contenido, educadores y profesionales que buscan mejorar su comunicación audiovisual.

  • Introducción a la interfaz y flujo de trabajo de Camtasia, Filmora y Sony Vegas (principales diferencias y similitudes).
  • Técnicas esenciales de edición: importar medios, cortar, recortar, unir clips, trabajar con la línea de tiempo y montaje básico.
  • Uso de transiciones fluidas, aplicación de efectos visuales básicos (filtros, ajustes de velocidad) y corrección de color fundamental.
  • Edición de audio: ajuste de niveles de volumen, añadir y sincronizar música de fondo, aplicar efectos de audio básicos (fade in/out) y reducción de ruido simple.
  • Incorporación de elementos gráficos: añadir textos, títulos estáticos y animados, insertar imágenes y logos.
  • Formatos de exportación (renderizado): configuración óptima para diferentes plataformas como YouTube, Instagram, TikTok y presentaciones.

Apoyo Académico en Matemáticas

Refuerza tus conocimientos y resuelve tus dudas en matemáticas. Ofrecemos clases personalizadas para diferentes niveles, desde aritmética básica hasta cálculo avanzado. Elige la modalidad que mejor se adapte a tus necesidades, por horas o por trabajo específico.

  • Temario Personalizado: Cubrimos temas como álgebra, geometría, trigonometría, estadística y cálculo diferencial e integral.
  • Modalidad por Horas: Sesiones de tutoría individual en vivo para reforzar conceptos, resolver ejercicios y preparar exámenes.
  • Modalidad por Trabajos: Ayuda en la resolución de deberes, proyectos o ejercicios específicos con entrega de soluciones detalladas.

Apoyo Académico en Física

Comprende los principios fundamentales de la física con la ayuda de un instructor. Ofrecemos tutorías para que domines la mecánica, la termodinámica, el electromagnetismo y más. Puedes contratar el servicio por sesiones de una hora o por trabajos específicos a entregar.

  • Temario Personalizado: Incluye mecánica clásica, cinemática, dinámica, termodinámica, óptica y electricidad.
  • Modalidad por Horas: Clases de apoyo enfocadas en la comprensión de las leyes físicas y la aplicación de fórmulas en problemas prácticos.
  • Modalidad por Trabajos: Asesoramiento para la elaboración de proyectos, informes de laboratorio o la resolución completa de guías de ejercicios.
