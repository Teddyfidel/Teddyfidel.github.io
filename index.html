Nuestra Oferta Educativa

Paquete Completo de Office

Domina Word, Excel y PowerPoint para optimizar tu productividad en cualquier entorno profesional.

  • Word: Formato avanzado, estilos, correspondencia.
  • Excel: Fórmulas, funciones, tablas dinámicas, gráficos.
  • PowerPoint: Diseño de diapositivas, animaciones, presentaciones efectivas.

Desarrollo Web Full Stack

Crea aplicaciones web interactivas desde cero aprendiendo HTML5, CSS3, JavaScript moderno y bases de datos SQL.

  • Maquetación semántica con HTML5.
  • Diseño responsive y moderno con CSS3 (Flexbox, Grid).
  • Programación frontend con JavaScript (ES6+).
  • Gestión de bases de datos relacionales con SQL.

Fundamentos de Programación

Iníciate en el mundo del código con Python, C++, SQL y Pseint, desarrollando una sólida lógica de programación.

  • Introducción a algoritmos y pseudocódigo (Pseint).
  • Sintaxis y estructuras básicas en Python y C++.
  • Consultas fundamentales en SQL.
  • Resolución de problemas y pensamiento lógico.

Manejo Experto de Windows

Optimiza tu uso del sistema operativo más popular: navegación, organización de archivos y configuraciones esenciales.

  • Explorador de archivos: Carpetas, vistas, búsqueda avanzada.
  • Panel de Control y Configuración del sistema.
  • Creación y edición de documentos básicos.
  • Consejos de seguridad y mantenimiento.

Herramientas de Inteligencia Artificial

Aprende a utilizar las IAs más potentes (ChatGPT, Copilot, Gemini, etc.) para potenciar tu creatividad y eficiencia.

  • Aplicaciones prácticas en estudios y trabajo.
